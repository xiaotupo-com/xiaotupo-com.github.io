反射（Reflection）定义：

Java 中的反射是指程序在运行期间可以拿到一个对象的所有信息处。

正常情况下调用一个对象的方法或访问一个对象的字段，通常会new一个对象的实例：

public class Main {

	public static void main(String[] args) throws Exception {
		// TODO Auto-generated method stub
		
        // 正常情况下创建一个对象
        Student stu = new Student("李平");
        String name = stu.getName();
	}
}

# Class 类

除了基本数据类型外（如：int等），Java 中的其他类型都是 Class ，就是在 JVM 类加载时会为没有中类型创建一个 Class 类型的对象，如：String 类型在加载时就会创建一个 String 的 Class 类型实例，并且不过创建多少个 String 类型的对象，都只会有一个 String 的 Class 类型实例。

一个 Class 实例会包含一个类的完整信息：

一个 String 例子

  • name = "java.lang.String"
  • package = "java.lang"
  • super = "java.lang.Object"
  • interface = ...
  • field = value[]...
  • method = ....

由于 Class 包含了对象的完整信息，所以我们可以通过 Class 来使用具体的对象的信息：类名、包名、父类、实现的接口、字段和方法等。

获取 Class 的方法有：

方法1：通过对象的静态变量 class 获取，如下：

Class stringClass = String.class;

方法2：通过对象实例的 getClass() 方法获取

String s = "Hello";
Class stringClass = s.getClass();

方法3：通过 Class 类的今天方法 Class.forName()，通过类的完整包路径

Class stringClass = Class.forName("java.lang.String");

获取实例的基本信息：

public class Main {
    public static void main(String[] args) {
        printClassInfo("".getClass());
        printClassInfo(Runnable.class);
        printClassInfo(java.time.Month.class);
        printClassInfo(String[].class);
        printClassInfo(int.class);
    }

    static void printClassInfo(Class cls) {
        System.out.println("Class name: " + cls.getName());
        System.out.println("Simple name: " + cls.getSimpleName());
        if (cls.getPackage() != null) {
            System.out.println("Package name: " + cls.getPackage().getName());
        }
        System.out.println("is interface: " + cls.isInterface());
        System.out.println("is enum: " + cls.isEnum());
        System.out.println("is array: " + cls.isArray());
        System.out.println("is primitive: " + cls.isPrimitive());
    }
}

# 访问字段

对任意的一个Object实例，只要我们获取了它的Class，就可以获取它的一切信息。

访问字段相关的 API ：

  • Field getField(name)：根据字段名获取某个public的field（包括父类）
  • Field getDeclaredField(name)：根据字段名获取当前类的某个field（不包括父类）私有
  • Field[] getFields()：获取所有public的field（包括父类）
  • Field[] getDeclaredFields()：获取当前类的所有field（不包括父类）私有

测试：

package com.xiaotupo.test;

import java.lang.reflect.Field;
import java.lang.reflect.Method;

import com.xiaotupo.Reflection;

public class ReflectionTest {

	public static void main(String[] args) throws Exception, ClassNotFoundException {
		// TODO Auto-generated method stub
	
		// 获取 Student1 的 class 对象
		Class stu1Class = Student1.class;
		
		Object object = stu1Class.getDeclaredConstructor().newInstance();
		
		// 获取 public 字段
		System.out.println(stu1Class.getField("score"));
		// 获取继承的public字段"name":
		System.out.println(stu1Class.getField("name"));
		// 获取private字段"grade":
		System.out.println(stu1Class.getDeclaredField("grade"));
		// 不能获取父类的 private字段"test": 将会报错，需要把下面一行注释起来
		//System.out.println(stu1Class.getDeclaredField("test"));
		
		
	} /* Main() END! */

}

class Student1 extends Person {
	public int score = 100;
	private int grade;
}

class Person {
	public String name;
	private int test;
}

输出：

public int com.xiaotupo.test.Student1.score
public java.lang.String com.xiaotupo.test.Person.name
private int com.xiaotupo.test.Student1.grade

# 修改字段的值

# 获取字段的信息

  • getName()：返回字段名称，例如，"name"；
  • getType()：返回字段类型，也是一个Class实例，例如，String.class；
  • getModifiers()：返回字段的修饰符，它是一个int，不同的bit表示不同的含义。
Field f = String.class.getDeclaredField("value");
f.getName(); // "value"
f.getType(); // class [B 表示byte[]类型
int m = f.getModifiers();
Modifier.isFinal(m); // true
Modifier.isPublic(m); // false
Modifier.isProtected(m); // false
Modifier.isPrivate(m); // true
Modifier.isStatic(m); // false

# testField.setAccessible(true);

package com.xiaotupo.test;

import java.lang.reflect.Field;
import java.lang.reflect.Method;

import com.xiaotupo.Reflection;

public class ReflectionTest {

	public static void main(String[] args) {
		// TODO Auto-generated method stub

	

		Class personClass = Person.class;
		
		Object object = personClass.getDeclaredConstructor().newInstance();
		
		Field testField = personClass.getDeclaredField("test");
		testField.setAccessible(true); // 执行这行后可以访问 private 字段
		Object testObject = testField.get(object);
		System.out.println("testObject: " + testObject);
		
		
	} /* Main() END! */

}

class Person {
	public String name;
	private int test = 1000;
}

调用Field.setAccessible(true)的意思是，别管这个字段是不是public，一律允许访问。

# 设置字段

通过反射设置字段使用 Field.set(Object, Object) 来实现，第一个 Object 参数是指要设置的对象， 第二个是要设置的值。

package com.xiaotupo.test;

import java.lang.reflect.Field;
import java.lang.reflect.Method;

import com.xiaotupo.Reflection;

public class ReflectionTest {

	public static void main(String[] args) throws Exception, ClassNotFoundException {
		// TODO Auto-generated method stub
	
		// 获取 Student1 的 class 对象
		Class stu1Class = Student1.class;
		Class personClass = Person.class;
		
		Object object = stu1Class.getDeclaredConstructor().newInstance();
		
		Field scoreField = stu1Class.getField("score");
		Object valueObject = scoreField.get(object);
		System.out.println("valueObject: " + valueObject);
		
		Field testField = personClass.getDeclaredField("test");
		testField.setAccessible(true);
		Object testObject = testField.get(object);
		System.out.println("testObject: " + testObject);
		
		// 设置 testField 字段 
		testField.set(object, 111);
		testObject = testField.get(object);
		System.out.println("testObject: " + testObject);
		
		
	} /* Main() END! */

}

class Student1 extends Person {
	public int score = 100;
	private int grade;
}

class Person {
	public String name;
	private int test = 1000;
}

同样的，修改非public字段，需要首先调用setAccessible(true)。

# 调用方法

调用方法和访问字段类似，都是通过指定的 API 来实现，调用方法的函数列表如下：

  • Method getMethod(name, Class...)：获取某个public的Method（包括父类）
  • Method getDeclaredMethod(name, Class...)：获取当前类的某个Method（不包括父类）
  • Method[] getMethods()：获取所有public的Method（包括父类）
  • Method[] getDeclaredMethods()：获取当前类的所有Method（不包括父类）

例子：

package com.xiaotupo.test;

public class ReflectionTest {

	public static void main(String[] args) {
		// TODO Auto-generated method stub
	
		// 获取 Student1 的 class 对象
		Class stu1Class = Student1.class;
		
		// 获取public方法getScore，参数为String:
        System.out.println(stu1Class.getMethod("getScore", String.class));
        // 获取继承的public方法getName，无参数:
        System.out.println(stu1Class.getMethod("getName"));
        // 获取private方法getGrade，参数为int:
        System.out.println(stu1Class.getDeclaredMethod("getGrade", int.class));
		
	} /* Main() END! */

}

class Student1 extends Person {
	public int getScore(String type) {
		return 100;
	}
	
	private int getGrade(int year) {
		return 1;
	}
}

class Person {

	public String getName() {
		return "Person";
	}
}

输出：

public int com.xiaotupo.test.Student1.getScore(java.lang.String)
public java.lang.String com.xiaotupo.test.Person.getName()
private int com.xiaotupo.test.Student1.getGrade(int)

一个对象包含的信息：

  • getName()：返回方法名称，例如："getScore"；
  • getReturnType()：返回方法返回值类型，也是一个Class实例，例如：String.class；
  • getParameterTypes()：返回方法的参数类型，是一个Class数组，例如：{String.class, int.class}；
  • getModifiers()：返回方法的修饰符，它是一个int，不同的bit表示不同的含义。
package com.xiaotupo.test;

import java.lang.reflect.Constructor;
import java.lang.reflect.Method;

public class ReflectionTest {

	public static void main(String[] args) throws Exception, ClassNotFoundException {
		// TODO Auto-generated method stub
	
		// 获取 Student1 的 class 对象
		Class<Student1> stu1Class = Student1.class;
		
        // 得到方法实例
		Method getScoreMethod = stu1Class.getMethod("getScore", String.class);
        // 调用方法
		Object score = getScoreMethod.invoke(stu1Class.getDeclaredConstructor().newInstance(), "");
		System.out.println("Score: " + score);
		
		
	} /* Main() END! */

}

class Student1 extends Person {
	public int getScore(String type) {
		return 100;
	}
	
	private int getGrade(int year) {
		return 1;
	}
}

class Person {

	public String getName() {
		return "Person";
	}
}

输出：

Score: 100

# 调用构造函数

通过类对象调用 newInstance() 方法，使用与无参构造函数，并且现在这个函数以及淘汰了，往后有 getConstructor().newInstance() 这样的函数代替，就是先获取对象的构造函数，然后在调用让后调用新版的newInstance()函数，并且这些函数都支持传参。

构造函数列表：

  • getConstructor() // 只可以用于有 public 修饰符的构造函数，默认构造函数要用 getDeclaredConstructor()
  • getDeclaredConstructor() // 获取"某个构造方法"可以是私有的，或受保护、默认、公有；
  • getConstructors() // 所有"公有的"构造方法
  • getDeclaredConstructors() // 获取所有的构造方法 包含私有的受保护的

忽略修饰符的方法：

如果设置 setAccessible(true) 为真，则忽略构造函数的修饰符，不管是 public、private 或 protected 都可以使用。

constructor.setAccessible(true);

# 获取继承关系

获取父类的 Class

package com.xiaotupo.test;

import java.lang.reflect.Constructor;
import java.lang.reflect.Method;

public class ReflectionTest {

	public static void main(String[] args) throws Exception {
		// TODO Auto-generated method stub
	
		// 获取 Student1 的 class 对象
		Class<Student1> stu1Class = Student1.class;
		
		Class<? super Student1> personClass = stu1Class.getSuperclass();
		
		Method getNameMethod = personClass.getMethod("getName");
		Object nameObject = getNameMethod.invoke(personClass.getDeclaredConstructor().newInstance());
		
		System.out.println("nameObject: " + nameObject);
		
		
	} /* Main() END! */

}

class Student1 extends Person {
	public int getScore(String type) {
		return 100;
	}
	
	private int getGrade(int year) {
		return 1;
	}
}

class Person {

	public String getName() {
		return "Person";
	}
}
