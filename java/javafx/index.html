JavaFx 是一个 Java 的 GUI 框架，用于开发桌面程序。

javafx 可以实现逻辑和样式的分离，可以使用xml和css来编写样式。

自从java11以后，jdk已经不内置javafx库，已交给开源社区管理，所以我们需要自己导入。

# Hello,World

JavaFX 中有一个 Application 类，我们可义创建一个继承自该类的类，让后重写 start() 函数，我们可以在 start() 函数中写上我们要显示的 Hello,World!。

package com.xiaotupo.demo;

import javafx.application.Application;
import javafx.fxml.FXMLLoader;
import javafx.scene.Scene;
import javafx.stage.Stage;


import java.io.IOException;

public class HelloApplication extends Application {
    @Override
    public void start(Stage stage) throws IOException {
        System.out.println("start() -------");
        System.out.println("start()：" + Thread.currentThread().getName());
        FXMLLoader fxmlLoader = new FXMLLoader(HelloApplication.class.getResource("hello-view.fxml"));
        Scene scene = new Scene(fxmlLoader.load(), 520, 240);
        stage.setTitle("Hello!");
        stage.setScene(scene);
        stage.show();
    }

    @Override
    public void init() throws Exception {
        super.init();

        System.out.println("init() -------");
        System.out.println("init()：" + Thread.currentThread().getName());
    }

    @Override
    public void stop() throws Exception {
        super.stop();

        System.out.println("stop() -------");
        System.out.println("stop()：" + Thread.currentThread().getName());

    }

    public static void main(String[] args) {
        launch();
    }
}
查看运行结果

# main() 类 和 Application 类分离

创建一个 MainWindow 类，并且继承 Application 类，重写 start() 方法，和在一个类中时一样，只不过该类中没有了 main() 函数。

程序是从 main() 入口函数开始的，所以我们还要在原来的类中来启动 javafx，我们在 HelloApplication.java 的main() 函数中调用 Application 抽象类的 launch() 静态函数来运行，这里传入的参数为 Application 的 class 类，也就是用到了 Java 的反射功能。

// FileName: MainWindow.java
package com.xiaotupo.demo;

import javafx.application.Application;
import javafx.fxml.FXMLLoader;
import javafx.scene.Scene;
import javafx.stage.Stage;

public class MainWindow extends Application {

    final String MAIN_WINDOW_TITLE = "我的第一个 JavaFX 程序";
    @Override
    public void start(Stage stage) throws Exception {
        stage.setTitle("MainWindow");

        // 创建 FXML 加载器
        FXMLLoader loader = new FXMLLoader(MainWindow.class.getResource("hello-view.    fxml"));
        // 创建场景
        Scene mainScene = new Scene(loader.load());

        stage.setTitle(MAIN_WINDOW_TITLE);
        stage.setScene(mainScene);
        stage.setMinWidth(350);
        stage.setMinHeight(200);
        stage.show();
    }
}


package com.xiaotupo.demo;

import javafx.application.Application;

public class HelloApplication {
    public static void main(String[] args) {
        Application.launch(MainWindow.class, args);
    }
}

        LauncherImpl.launchApplication(var0, var1);
}

# JavaFx 窗口分层结构

JavaFx 窗口的最外层是 Stage ，一个 Stage 就是一个独立的窗口。

往里第二层是 Scene(场景)，一个 Stage 中可以有多个场景。

Scene 里面是各个节点，就是具体的内容，如按钮，文本框，输入框等。

# JavaFx 窗口

在 JavaFx 中一个窗口就是有 Stage 类创建的实例，如下：

// 部分代码
@FXML
protected void onHelloButtonClick() {
    welcomeText.setText("Welcome to JavaFX Application!");

    Stage s1 = new Stage(); // 创建一个窗口 Stage
    /* 设置 s1 窗口的属性 */
    s1.setTitle("测试窗口");
    s1.setHeight(200);
    s1.setWidth(400);
    s1.setMinHeight(200);
    s1.setMinWidth(300);

    Group group = new Group();
    Scene sc1 = new Scene(group);

    s1.setScene(sc1); // 设置窗口的场景为 sc1

    s1.show();

}

# 监听事件

我们来看看 JavaFx 是怎么监听事件的，来看几个简单的例子。

坐标：上下位 Y 轴，左右位 X 轴，左上角为零点(x: 0, y: 0)。

// ========================= 监听事件 =========================
stage.heightProperty().addListener(new ChangeListener<Number>() {
    @Override
    public void changed(ObservableValue<? extends Number> observableValue, Number number, Number t1) {
        System.out.printf("height::observableValue:: %s\n" +
                "height::number:: %s\n" +
                "height::t1:: %s\n", observableValue.getValue(), number, t1

        );
    }
});

stage.widthProperty().addListener(new ChangeListener<Number>() {
    @Override
    public void changed(ObservableValue<? extends Number> observableValue, Number number, Number t1) {
        // observableValue: 抽象接口，可以使用 .getValue() 获取窗口现在宽度
        // number: 窗口原来宽度
        // t1: 窗口现在宽度
        System.out.printf("width::observableValue:: %s\n" +
                "width::number:: %s\n" +
                "width::t1:: %s\n", observableValue.getValue(), number, t1
        );
    }
});

// ========================= 监听事件结尾 =========================
