# 强制类型转换

有时我们需要进行强制类型转换，语法为在括号(short) 中写上要转换成什么类型的数据，下面是一个 int 转换为 short 类型的例子。

public class Hello {
    public static void main(String[] args) {

        int a = 1000000;
        short b = (short)a;
        System.out.println(b); // Output: 16960

        System.out.println("Hello Java!");
    }
}

String 不是基本数据类型，而是 引用 数据类型。

# 把字符串转换为其他类型

public class Hello {
    public static void main(String[] args) {
        String s = "123";
        int i = Integer.parseInt(s);
        float f = Float.parseFloat(s);
        double df = Double.parseDouble(s);
        short si = Short.parseShort(s);
        boolean b = Boolean.parseBoolean(s);

        System.out.format("s: %s\n", s);
        System.out.format("i: %d\n", i);
        System.out.format("f: %f\n", f);
        System.out.format("df: %f\n", df);
        System.out.format("si: %d\n", si);
        System.out.format("b: %b\n", b);

        System.out.printf("s: %s\ni: %d\nf: %f\ndf: %f\nsi: %d\nb: %b\n", s, i, f, df, si, b); // printf() 函数演示
        System.out.println("Hello Java!");
    }
}

# 数组

创建数组的两种方式：

  • int[] nums = {1, 2, 3, 4, 5}; // 创建一个数组
  • int[] sizes = new int[10]; // 创建一个数组
public class Hello {
    public static void main(String[] args) {
        int[] nums = {1, 2, 3, 4, 5}; // 创建一个数组
        int[] sizes = new int[10];  // 创建一个数组


        for (int i = 0; i < sizes.length; i++) {
            sizes[i] = i;
        }

        for (int i = 0; i < sizes.length; i++) {
            System.out.printf("sizes[%d]: %d\n", i, sizes[i]);
        }

        for (int v : nums) {
            System.out.printf("ak: %d\n", v);
        }

        System.out.println("Hello Java!");
    }
}

# 数组作为参数

public class Hello {

    public static void main(String[] args) {
        int[] nums = {1, 2, 3, 4, 5}; // 创建一个数组
        int[] sizes = new int[10];  // 创建一个数组


        for (int i = 0; i < sizes.length; i++) {
            sizes[i] = i;
        }

        printIntArray(nums);    // 打印数组
        printIntArray(sizes);   // 打印数组
        printIntArray(new int[]{100, 200, 300}); // 打印数组

        System.out.println("Hello Java!");
    }

    // 数组作为函数参数
    private static void printIntArray(int[] array) {
        for (int i = 0; i < array.length; i++) {
            System.out.printf("array[%d]: %d\n", i, array[i]);
        }
        System.out.printf("------------------- 分割线 -------------------\n");
    }

}

# 数组作为函数返回值

public class Hello {

    public static void main(String[] args) {
        int[] nums = {1, 2, 3, 4, 5}; // 创建一个数组
        int[] sizes = new int[10];  // 创建一个数组


        for (int i = 0; i < sizes.length; i++) {
            sizes[i] = i;
        }

        printIntArray(nums);    // 打印数组
        printIntArray(sizes);   // 打印数组
        printIntArray(new int[]{100, 200, 300}); // 打印数组

        int[] newarray = getNewArray();
        printIntArray(newarray);

        System.out.println("Hello Java!");
    }

    private static void printIntArray(int[] array) {
        for (int i = 0; i < array.length; i++) {
            System.out.printf("array[%d]: %d\n", i, array[i]);
        }
        System.out.printf("------------------- 分割线 -------------------\n");
    }

    // 数组作为函数返回值
    private static int[] getNewArray() {
        return new int[]{-100, -100, -100, -100};
    }
}

# Arrays 类

java.util.Arrays 类能方便地操作数组，它提供的所有方法都是静态的。

如 fill 静态方法，例子如下：

public class Hello {
    public static void main(String[] args) {
        int[] nums = {1, 2, 3, 4, 5}; // 创建一个数组
        int[] sizes = new int[10];  // 创建一个数组

        Arrays.fill(nums, 10); // 将数组 `nums` 中的每一项都改为 10
        printIntArray(nums);

        System.out.println("Hello Java!");
    }
}

Arrays 类中还有很多其他方法，可使用时在做了解。

# List 泛型类

List 是一种泛型类，也就是可以通过它定义不同类型的数组列表，创建 List 的方法有多种，如：ArrayList、LinkedList和List.of()都可以。

package com.example.hello_java;


import java.util.ArrayList;
import java.util.Arrays;
import java.util.List;

public class Hello {

    public static void main(String[] args) {
        List<String> names = new ArrayList<>();
        names.add("张三");
        names.add("李子");
        String z = names.remove(0); // 从 names 中删除 索引为 0 的元素，并返回
        System.out.printf("z: %s\n", z);

        System.out.printf("%s\n", Arrays.toString(names.toArray()));

    }

}
