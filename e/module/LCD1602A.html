LCD1602A 是单片机开发中常用的一种显示屏，可以显示简单的字母、数字、符号等内容。可以显示的字符数量为 16x2，也就是可以显示两行，每行16个字符。

LCD1602A 的数据传输方式可以分为4位和8位两种，每块显示区域提供 5x7 个点阵+游标的显示模式。提供显示数据缓冲区 DDRAM、字符发生器CGROM和字符发生器CGRAM，可以用CGRAM 来存储自己定义的限多8个5x8点阵的图形字符的字模数据。提供了丰富的指令：清除显示、游标回原点、显示开关、游标开关、显示字符闪烁、游标移位、显示移位元等。提供内部上电自动复位电路，当外加电源电压超过+4.5V时，自动对模块进行初始化操作，将模块设置为默认的显示工作状态。

一些引脚的介绍：

  • VL引脚：为液晶显示器对比度调整端，接正电源时对比度最弱，接地时对比度最高，对比度过高时会产生“鬼影”，使用时可以通过一个10K的电位器调整对比度；
  • RS引脚：RS为寄存器选择，高电平时选择数据寄存器、低电平时选择指令寄存器
  • R/W引脚：R/W为读写信号线，高电平时进行读操作，低电平时进行写操作。当RS和R/W共同为低电平时可以写入指令或者显示地址，当RS为低电平R/W为高电平时可以读忙信号，当RS为高电平R/W为低电平时可以写入数据。
  • EN引脚：E端为使能端，当E端由高电平跳变成低电平时，液晶模块执行命令。

# 写数据

void write_data(uint8_t date)
{
    LCDRS = 1;
    P0 = date;
    LCDDelay(5);
    LCDEN = 1;
    LCDDelay(5);
    LCDEN = 0;
}

# 写指令

void write_com(uint8_t com)
{
    LCDRS = 0;
    P0 = com;
    LCDDelay(5);
    LCDEN = 1;
    LCDDelay(5);
    LCDEN = 0;
}

# 初始化函数

LCD1602 初始化需要：

  1. 设置 R/W 为低电平，低电平为写模式
  2. 屏幕设置模式：写指令 38H
  3. 打开显示，关闭光标：写指令 0CH
  4. 设置显示光标移动位置：写指令 06H
  5. 设置字符显示位置：写指令（字符显示的起始位置）
void init1602()
{
    uint8_t i = 0;
    LCDRW = 0;
    write_com(0x38); // 显示屏初始化
    write_com(0x0c); // 打开显示 无光标 无闪烁
    write_com(0x06); // 当读或写一个字符是指针后一位
    write_com(0x01); // 清屏
    write_com(0x80); // 设置位置
}

# 指令列表

# 1、清屏

RSRWDB7DB6DB5DB4DB3DB2DB1DB0
0000000001

运行时间（250KHz); 1.64us 功能：清除DDRAM 和 AC值

# 2、归为

RSRWDB7DB6DB5DB4DB3DB2DB1DB0
000000001*

运行时间（250KHz); 1.64us 功能：AC=0, 光标，画面会 HOME 位

# 3、输入方式设置

RSRWDB7DB6DB5DB4DB3DB2DB1DB0
00000001I/DS

运行时间（250KHz); 1.64us 功能：设置光标，画面移动方式 其中：

  • I/D=1 数据读、写操作后， AC自动增一
  • I/D=0 数据读、写操作后，AC自动减一
  • S=1 数据读、写操作后，画面平移
  • S=0 数据读、写操作后，画面不变

# 4、显示开关控制

RSRWDB7DB6DB5DB4DB3DB2DB1DB0
0000001DCB

运行时间（250KHz); 1.64us 功能：设置显示，光标及闪烁开、关 其中：

  • D 表示显示开关：D=1为开，D=0为关
  • C 表示光标开关，C=1为开，C=0为关
  • B 表示闪烁开关，B=1为开，B=0为关

# 5、光标、画面位移

RSRWDB7DB6DB5DB4DB3DB2DB1DB0
000001S/CR/L**

运行时间（250KHz); 1.64us 功能：光标、画面移动，不影响DDRAM 其中：

  • S/C=1 : 画面平原一个字符
  • S/C=0 : 光标平移一个字符位
  • R/L=1 : 右移， R/L=0 : 左移

# 6、功能设置

RSRWDB7DB6DB5DB4DB3DB2DB1DB0
00001DLNF**

运行时间（250KHz); 1.64us 功能：工作方式设置（初始化指令） 其中：

  • DL=1 设置为 8 位数据接口
  • DL=0 设置为 4 位数据接口
  • N=1 两行显示
  • N=0 一行显示
  • F=1 5X10点阵字符
  • F=0 5X7 点阵字符

# 7、CGRAM 地址设置

RSRWDB7DB6DB5DB4DB3DB2DB1DB0
0001A5A4A3A2A1A0

运行时间（250KHz); 1.64us 功能：设置CGRAM地址，A5-A0 = 0~3FH

# 8、DDRAM 地址设置

RSRWDB7DB6DB5DB4DB3DB2DB1DB0
001A6A5A4A3A2A1A0

运行时间（250KHz); 1.64us 功能：设置DDRAM地址

  • N=0, 一行显示 A6~A0 = 0~4FH
  • N=1,两行显示 首行A6~A0 = 00H~2FH, 次行A6~A0 = 40H~67H

# 9、读BF及AC值

RSRWDB7DB6DB5DB4DB3DB2DB1DB0
01BFAC6AC5AC4AC3AC2AC1AC0

运行时间（250KHz); 1.64us 功能：都忙BF值和地址计数器AC值 其中：

  • BF=1 忙
  • BF=0 准备好，此时 AC 值意义为最近一次地址设置（CGRAM或DDRAM）定义

# 10、写数据

RSRWDB7DB6DB5DB4DB3DB2DB1DB0
10D7D6D5D4D3D2D1D0

运行时间（250KHz); 1.64us 功能：根据最近设置的地址性质，数据写入DDRAM或CGRAM中

# 11、读数据

RSRWDB7DB6DB5DB4DB3DB2DB1DB0
11D7D6D5D4D3D2D1D0

运行时间（250KHz); 1.64us 功能：根据最近设置的地址性质，从DDRAM或CGRAM读数据
