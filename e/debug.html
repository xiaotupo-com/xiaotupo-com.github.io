# 维修中最容易坏的器件

  • 电容
  • 电阻
  • 二极管
  • 继电器
  • 电源芯片
  • 按键
  • 开关
  • 光耦
  • 运放

# 电容

贴片电容，贴片电容损害时表现为：

  • 短路：短路时电容的电阻值几乎为0，电路无法工作
  • 非完全短路：阻止非常低，可能在一百欧姆左右，电容发热严重
  • 容值变低：使滤波效果变低，有可能使电路工作不稳定

在电源电路中电容发生损坏的几率比较高。

# 电阻

  • 电阻开路：会使电路完全不通
  • 阻值严重偏离标准值，偏低或高

电阻损坏会使电路限流或分压电路不准，使电路工作在不正常的状态

# 二极管

  • 击穿短路：二极管最常见的损坏就是击穿电路了，会使电路无法工作，发热严重甚至会把电路板烧毁
  • TVS 称瞬变电压抑制二极管在 RS485 保护电路中如果漏电流变大可能会影响通信，需要更换新的 TVS管

# 继电器

继电器发生损坏的情况为：

  • 触电粘连：常开触点无法断开
  • 线圈短路：线圈短路使发热严重，可能会使控制继电器的三极管或MOS管烧掉

# 电源芯片

电源芯片常见的问题有：

  • 芯片内部损坏，使输出电压不准，需要更换新的
  • 芯片短路，可能芯片会炸裂，也会使其他保护器件损坏，如：保险、TVS、PCB铜箔等

# 按键

按键使用久了会不灵，需要更换新的

# 开关

有时会发生电路供电偶尔不供电或不稳定，就很有可能是开关的问题，首先要确定开关无问题

# 光耦

光耦是控制类器件，损坏时会无法控制，可以用万用表量是否有控制信号，让后看输出的是否可以完全截止和导通
