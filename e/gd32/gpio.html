# GPIO 寄存器

GPIO 存储器映射地址范围：

  • GPIOA 基地址：0x4800 0000
  • GPIOB 基地址：0x4800 0400
  • GPIOC 基地址：0x4800 0800
  • GPIOD 基地址：0x4800 0C00
  • GPIOF 基地址：0x4800 1400

端口在官方库中的定义：

gd32f1x0_gpio.h

#define GPIOA                      (GPIO_BASE + 0x00000000U)
#define GPIOB                      (GPIO_BASE + 0x00000400U)
#define GPIOC                      (GPIO_BASE + 0x00000800U)
#define GPIOD                      (GPIO_BASE + 0x00000C00U)
#define GPIOF                      (GPIO_BASE + 0x00001400U)

相关存储映射定义 gd32f1x0.h

#define AHB2_BUS_BASE         ((uint32_t)0x48000000U)        /*!< ahb2 base address
#define GPIO_BASE             (AHB2_BUS_BASE + 0x00000000U)  /*!< GPIO base address

# GPIO 寄存器列表

  1. 控制寄存器（GPIOx_CTL）
  2. 输出模式寄存器（GPIOx_OMODE）
  3. 输出速度寄存器（GPIOx_OSPD）
  4. 上/下拉寄存器（GPIOx_PUD）
  5. 端口输入状态寄存器（GPIOx_ISTAT, x=A..D,F）
  6. 端口输出控制寄存器（GPIOx_OCTL, x=A..D,F）
  7. 端口位操作寄存器（GPIOx_BOP, x=A..D,F）
  8. 端口配置锁定寄存器（GPIOx_LOCK, x=A,B）
  9. 备用功能选择寄存器 0（GPIOx_AFSEL0, x=A,B,C）
  10. 备用功能选择寄存器 1（GPIOx_AFSEL1, x=A,B,C）
  11. 位清除寄存器（GPIOx_BC, x=A..D,F）
  12. 端口位翻转寄存器（GPIOx_TG, x=A..D,F）（仅适用于 GD32F170xx 和 GD32F190xx 产品）

# 控制寄存器（GPIOx_CTL）

TIP

  • 地址偏移：0x00
  • 复位值：端口 A 0x2800 0000；其他端口 0x0000 0000
  • 该寄存器可以按字节（8 位）、半字（16 位）或字（32 位）访问。

# CTL0[1:0] Pin0 配置

TIP

  • 该位由软件置位和清除。
  • 00：输入模式（复位值）
  • 01：GPIO 输出模式
  • 10：备用功能模式
  • 11：模拟模式

其他 Pin 的配置和上面一样，参考上面配置即可。

官方库宏定义

#define GPIO_CTL(gpiox)            REG32((gpiox) + 0x00000000U)

使用方法：

配置 `PA0` 为输出模式：
GPIO_CTL(GPIOA) &= ~GPIO_CTL_CTL0; // GPIO_CTL_CTL0 按位取反后为 1111 1111 1111 1100, 再与 GPIOA 的控制寄存器相与把 CTL0[1:0] 设置为 00
GPIO_CTL(GPIOA) |= 0x01; // GPIOA 的控制寄存器与 0x01 按位或运算，把 CTL0[1:0] 设置为 01
GPIO_CTL(GPIOA) |= GPIO_MODE_OUTPUT; // 这行代码和它上面以后作用相同，GD官方库中已经提供了一些实用的宏定义给我们使用

上面代码中的 GPIO_CTL_CTL0 宏定义如下：

#define GPIO_CTL_CTL0              BITS(0,1)

输出模式定义：

/* output mode definitions */
#define CTL_CLTR(regval)           (BITS(0,1) & ((uint32_t)(regval) << 0))
#define GPIO_MODE_INPUT            CTL_CLTR(0)           /*!< input mode */
#define GPIO_MODE_OUTPUT           CTL_CLTR(1)           /*!< output mode */
#define GPIO_MODE_AF               CTL_CLTR(2)           /*!< alternate function mode */
#define GPIO_MODE_ANALOG           CTL_CLTR(3)           /*!< analog mode */

gpio 模式设置可参考官方库 gpio_mode_set()函数，在gd32f1x0_gpio.c 中。

# 输出模式寄存器（GPIOx_OMODE）

TIP

  • 地址偏移：0x04
  • 复位值：0x0000 0000
  • 该寄存器可以按字节（8 位）、半字（16 位）或字（32 位）访问。

# OM0 Pin 0 输出模式位

该位由软件置位和清除。

  • 0：输出推挽模式（复位值）
  • 1：输出开漏模式

官网库宏定义：

#define GPIO_OMODE(gpiox)          REG32((gpiox) + 0x00000004U) // 获取 GPIO 输出模式寄存器地址

使用方法：

// 设置 PA0 为开漏输出模式
GPIO_OMODE(GPIOA) &= ~GPIO_OMODE_OM0;
GPIO_OMODE(GPIOA) |= 0x01;
GPIO_OMODE(GPIOA) |= GPIO_OTYPE_OD; // 和上一行代码作用相同，GPIO_OTYPE_OD 是官网提供的一个宏 值为 0x01

GPIO_OMODE_OM0 定义：

#define GPIO_OMODE_OM0             BIT(0)

输出模式:

/* GPIO output type */
#define GPIO_OTYPE_PP              ((uint8_t)(0x00U))    /*!< push pull mode */
#define GPIO_OTYPE_OD              ((uint8_t)(0x01U))    /*!< open drain mode */

GPIO 输出模式设置可参考官方库的 gpio_output_options_set() 函数，该函数在 在gd32f1x0_gpio.c 中。

# 输出速度寄存器（GPIOx_OSPD）

TIP

  • 地址偏移：0x08
  • 复位值：端口 A 0x0C00 0000；其他端口 0x0000 0000
  • 该寄存器可以按字节（8 位）、半字（16 位）或字（32 位）访问。

# OSPD0[1:0] Pin 0 输出最大速度位

该位由软件置位和清除。

  • x0：输出最大速度 2M（复位值）
  • 01：输出最大速度 10M
  • 11：输出最大速度 50M

官网库宏定义：

#define GPIO_OSPD(gpiox)           REG32((gpiox) + 0x00000008U)

使用方法：

// 配置输出最大速度为：50M
GPIO_OSPD(GPIOA) &= ~GPIO_OSPD_OSPD0;
GPIO_OSPD(GPIOA) |= GPIO_OSPEED_50MHZ; // 可以用 0x03 代替 GPIO_OSPEED_50MHZ

GPIO_OSPD_OSPD0 定义：

#define GPIO_OSPD_OSPD0            BITS(0,1)

最大输出速度宏定义：

/* GPIO output max speed value */
#define OSPD_OSPD(regval)          (BITS(0,1) & ((uint32_t)(regval) << 0))
#define GPIO_OSPEED_2MHZ           OSPD_OSPD(0)          /*!< output max speed 2M */
#define GPIO_OSPEED_10MHZ          OSPD_OSPD(1)          /*!< output max speed 10M */
#define GPIO_OSPEED_50MHZ          OSPD_OSPD(3)          /*!< output max speed 50M */

官方库函数 gpio_output_options_set()，可以同时配置输出模式和最大输出速度。

# 上拉/下拉寄存器（GPIOx_PUD）

TIP

  • 地址偏移：0x0C
  • 复位值：端口 A 0x2400 0000；其他端口 0x0000 0000
  • 该寄存器可以按字节（8 位）、半字（16 位）或字（32 位）访问。

# PUD0[1:0] Pin 0 上拉或下拉位

该位由软件置位和清除。

  • 00：悬空模式，无上拉和下拉（复位值）
  • 01：端口上拉模式
  • 10：端口下拉模式
  • 11：保留

官网库宏定义：

#define GPIO_PUD(gpiox)            REG32((gpiox) + 0x0000000CU)

使用方法：

// 设置 PA0 为上拉模式
GPIO_PUD(GPIOA) &= ~GPIO_PUD_PUD0;
GPIO_PUD(GPIOA) |= 0x01;
GPIO_PUD(GPIOA) |= GPIO_PUPD_PULLUP; // 和上行代码作用相同

GPIO_PUD_PUD0 宏定义：

#define GPIO_PUD_PUD0              BITS(0,1)

上下拉宏定义：

/* pull-up/pull-down definitions */
#define PUD_PUPD(regval)           (BITS(0,1) & ((uint32_t)(regval) << 0))
#define GPIO_PUPD_NONE             PUD_PUPD(0)           /* 0000 0000 0000 0000 */
#define GPIO_PUPD_PULLUP           PUD_PUPD(1)           /* 0000 0000 0000 0001 */
#define GPIO_PUPD_PULLDOWN         PUD_PUPD(2)           /* 0000 0000 0000 0010 */

# 配置锁定寄存器（GPIOx_LOCK, x=A,B）

TIP

  • 地址偏移：0x1C
  • 复位值：0x0000 0000
  • 该寄存器只能按字（32 位）访问。

# LKK 锁定键

该位只能通过 Lock Key 写序列置位，始终可读。

  • 0：GPIOx_LOCK 寄存器和端口配置没有锁定
  • 1：直到下一次 MCU 复位前，GPIOx_LOCK 寄存器被锁定

LOCK key 写序列： 写 1→写 0→写 1→读 0→读 1 注意：在 LOCK Key 写序列期间，LK y(y=0..15)的值必须保持。

# LKy 端口锁定位 y(y=0..15)

该位由软件置位和清除。

  • 0：端口配置没有锁定
  • 1：端口配置锁定

# 备用功能选择寄存器 0（GPIOx_AFSEL0, x=A,B,C）

TIP

  • 地址偏移：0x20
  • 复位值：0x0000 0000
  • 该寄存器可以按字节（8 位）、半字（16 位）或字（32 位）访问。

# SEL0[3:0] Pin 0 选择备用功能

该位由软件置位和清除。

  • 0000：选择 AF0 功能（复位值）
  • 0001：选择 AF1 功能
  • 0010：选择 AF2 功能
  • 0011：选择 AF3 功能
  • 0100：选择 AF4 功能（Port A，B only）
  • 0101：选择 AF5 功能（Port A，B only）
  • 0110：选择 AF6 功能（Port A，B only）
  • 0111：选择 AF7 功能（Port A，B only）
  • 1000：保留
  • 1001：选择 AF9 功能（Port A，B only）（仅适用于 GD32F170xx 和 GD32F190xx 产品）
  • 1010：保留
  • 1011：选择 AF11 功能（仅适用于 GD32F170xx 和 GD32F190xx 产品）
  • 1100 ~ 1111：保留

# 备用功能选择寄存器 1（GPIOx_AFSEL1, x=A,B,C）

TIP

  • 地址偏移：0x24
  • 复位值：0x0000 0000
  • 该寄存器可以按字节（8 位）、半字（16 位）或字（32 位）访问。

# SEL8[3:0] Pin 8 选择备用功能

该位由软件置位和清除。

  • 0000：选择 AF0 功能（复位值）
  • 0001：选择 AF1 功能
  • 0010：选择 AF2 功能
  • 0011：选择 AF3 功能
  • 0100：选择 AF4 功能（Port A，B only）
  • 0101：选择 AF5 功能（Port A，B only）
  • 0110：选择 AF6 功能（Port A，B only）
  • 0111：选择 AF7 功能（Port A，B only）
  • 1000：保留
  • 1001：选择 AF9 功能（Port A，B only）（仅适用于 GD32F170xx 和 GD32F190xx产品）
  • 1010：保留
  • 1011：选择 AF11 功能（仅适用于 GD32F170xx 和 GD32F190xx 产品）
  • 1100 ~ 1111：保留

# 位清除寄存器（GPIOx_BC, x=A..D,F）

TIP

  • 该寄存器可以按字节（8 位）、半字（16 位）或字（32 位）访问。
