今天介绍的消抖方法是利用 gd32 的 1ms 滴答定时器进行按键的非阻塞消抖，具体方法为：

  1. 在 SysTick_Handler 抵达定时器处理函数中定义一个 32位的 static 无符号变量作为32次按键的采集缓存
  2. 每1ms 就让 key_buf 缓存左移一位，并把当前按键的状态读到 key_buf 的最低位。
  3. 判断 key_buf 的每一位是否都为 0 或 1，全0代表按键按下，全1代表按键没按下， 如果不是全0或全1则代表都得不处理
  4. 根据自己的需求来处理 全0 或 全1的状态

# 代码

void SysTick_Handler(void)
{
	
	/* 1sm 执行一次 */
	static uint32_t key_buf = 0xffffffff; /* 按键缓存区 */
	FlagStatus key_state = USER_KEY_STATE; /* 读取按键当前状态 */
	static uint8_t ok_down_flag = 0;	/* 用户判断是成功按下后 */
	
	key_buf = (key_buf << 1) | key_state; /* key_buf 左移一位，并或上按键状态 key_state */
	
	/* 如果 32sm 内 32次读到的状态都为 0 代表按键按下了 */
	if (key_buf == (uint32_t)0x00000000)
	{
		user_key.current_state = RESET; /* 把状态写入全局变量 */
		ok_down_flag = 1;
	}
	/* 如果 flag 等于 1，并且 32 次读到的状态都为 1，代表按键弹起 */
	else if ((key_buf == (uint32_t)0xffffffff) && (ok_down_flag == 1))
	{
		user_key.current_state = SET; /* 把状态写入全局变量 */
		user_key.click_state = TRUE; /* 按键单击成功，把状态写入全局变量 */
		ok_down_flag = 0; /* 清除标志位 */
	} else {
		/* 抖动 */
		user_key.click_state = FALSE;  /* 按键单击失败 */
	}
	
}
